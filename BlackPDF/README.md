# BlackPDF - 颜色翻转PDF阅读器

一个支持颜色翻转的PDF阅读器，提供类似"暗黑模式"的阅读体验，将白色背景转换为黑色，黑色文字转换为白色。

## 功能特点

- 📖 **PDF阅读**: 支持打开和浏览PDF文件
- 🌙 **颜色翻转**: 一键切换暗黑模式，保护眼睛
- 🔍 **缩放功能**: 支持放大缩小，调整阅读舒适度
- 📄 **页面导航**: 上一页、下一页、跳转到指定页
- 🎨 **简洁界面**: 清晰直观的用户界面

## 操作说明

1. **打开PDF**: 点击"📁 打开PDF"按钮选择PDF文件
2. **翻页**: 使用"⬅️ 上一页"和"下一页 ➡️"按钮，或直接输入页码跳转
3. **缩放**: 使用"🔍-"和"🔍+"按钮调整显示大小
4. **切换模式**: 点击"🌙 暗黑模式"按钮在普通模式和颜色翻转模式之间切换

## 系统要求

- Python 3.6+
- PyQt5
- PyMuPDF (fitz)

## 技术说明

该工具使用PyMuPDF渲染PDF页面为图像，然后通过QImage的 `invertPixels()`方法实现颜色翻转，将：

- 白色(255,255,255) → 黑色(0,0,0)
- 黑色(0,0,0) → 白色(255,255,255)
- 其他颜色也会相应反转

这样可以在夜间或弱光环境下更舒适地阅读PDF文档。
